# MRA Image & Video App (app4.py)

## Requirements

- Python 3.10
- Conda (recommended)
- Packages (via `environment.yml in the root project`):
  - numpy
  - scipy
  - pillow
  - matplotlib
  - imageio
  - opencv-contrib-python
  - pywt (PyWavelets)
  - streamlit
  - pydicom + jpeg plugins (for DICOM)

# create 
conda activate smarttek2

# run the Streamlit app
streamlit run app4.py
